# PNG asset update

All SVG assets and SVG references have been removed from this project.

## Required logo paths

- Main website logo: `public/logo/logo.png`
- Footer Elix Code logo: `public/logo/Elix-logo.png`

## Additional changes

- Course and blog illustrations were converted to PNG.
- The web manifest icon now uses `public/icon.png`.
- UI icons are PNG files under `public/icons`.
- The `lucide-react` dependency was removed to prevent runtime SVG icon output.
